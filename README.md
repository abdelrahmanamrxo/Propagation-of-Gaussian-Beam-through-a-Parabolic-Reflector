# Propagation of Gaussian Beam through a Parabolic Reflector
This MATLAB code is submitted as the course project for the "ECE311s -  Electromagnetic Waves" course given at the Faculty of Engineering, Ain Shams University.

It covers simulating a Gaussian Beam through space before falling on a parabolic reflector.

The documentation can be found [here](https://drive.google.com/file/d/1_tSJHu2gm1ApAjU3uV1Um3IMWzVNEItv/view?usp=drive_link).
